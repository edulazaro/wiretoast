API (igual en los 3 entornos, siempre notify)

JavaScript (vanilla)

window.notify('Cambios guardados', 'success');
window.notify({ title: 'Guardado', message: 'Todo correcto' }, 'success');
window.notify('Texto', 'warning', { position: 'bottom-center', timeout: 3000 });

Alpine.js

<button @click="$dispatch('notify', { message: 'Hola', type: 'success' })">
    Trigger
</button>

<button @click="$dispatch('notify', {
    title: 'Guardado',
    message: 'Todo OK',
    type: 'success',
    position: 'bottom-right'
})">Con título</button>

Livewire (PHP)

// Texto simple
$this->notify('Cambios guardados', 'success');

// Con título
$this->notify(['title' => 'Guardado', 'message' => 'Todo OK'], 'success');

// Con opciones
$this->notify('Mensaje', 'warning', [
    'position' => 'bottom-center',
    'timeout'  => 3000,
]);

Tipos: success, error, warning, info, neutral. Tema: data-wt-theme en <body>. Modo: data-wt-theme-mode. Posición por defecto: data-wt-default-position.

Internamente, tanto Livewire como Alpine disparan un CustomEvent notify en window, que el componente escucha.
